Smart Weather Station with ESP32

This project demonstrates the integration of sensors with the ESP32 to create a smart weather station. It logs data from the BMP280 and LSM9DS1 sensors onto an SD card, displays real-time data on an OLED screen, and includes Wi-Fi connectivity for cloud monitoring.

Project Overview

Combining both hardware and software, this project is designed to be scalable and modular. It lays the foundation for integrating additional sensors and advanced data analytics features.

Technologies Used

  • ESP32 Microcontroller
  • BMP280 Sensor (Pressure & Temperature)
  • LSM9DS1 Sensor (Accelerometer & Gyroscope)
  • SD Card Module
  • OLED Display
  • Wi-Fi Connectivity for Cloud Integration
