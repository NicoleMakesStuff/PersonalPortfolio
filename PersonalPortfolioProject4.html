GPS-Based Location Tracker with ESP32

This project features a GPS-based tracker built with an ESP32 and the TinyGPS++ library. It logs real-time location data to EEPROM and an SD card, while offering a web-based interface to view routes on a map.

Project Overview

The tracker is designed to monitor and store location data seamlessly. The web interface integrates with Google Maps API to visualize tracked routes, providing a practical solution for location-based tracking.

Technologies Used

  • ESP32 Microcontroller
  • TinyGPS++ Library
  • EEPROM & SD Card Storage
  • Google Maps API
  • HTML, CSS & JavaScript for Web Interface
